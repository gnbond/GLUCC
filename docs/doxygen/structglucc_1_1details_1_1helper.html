template<typename T, typename S, typename U = void>
struct glucc::details::helper< T, S, U >

Base template.

This is the base template, and it establishes two things:

  • Any 2-argument instantiation helper<T,S> is actually instantiating helper<T,S,void>. The application of default template arguments happens very early in the template instantiation process, before any possible specializations are considered
  • The default answer for helper is false_type
